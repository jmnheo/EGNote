# Project: [스프레드시트 데이터를 조회하는 웹페이지]

## language
- 모든 설명 및 커뮤니케이션은 한국어로 진행한다.

## Overview
이 프로젝트는 간단한 웹 기반 도구를 개발하는 사이드 프로젝트입니다.
매일 반복되는 특정 프로필 목록을 조회하고, 상세 화면에서 특이사항을 확인하는 용도로 사용됩니다. 예시: 투석 전문 병원에서 아침 회진시 반복적으로 사용하는 환자 차트

## Tech Stack
- allowed: 
    React 사용 가능
    Tailwind CSS 사용 가능
    shadcn/ui 사용 가능
    TypeScript 선택 사용 가능
- Preferred Stack:
    React
    Tailwind CSS
    shadcn/ui
    Vite

## UI Rules
- Mobile First 디자인
- iPhone Safari 호환성 우선 고려
- 단순한 UI 유지
- 터치 인터랙션 최적화

## Code Principles
- 단순성 우선
- 작은 파일 구조 유지
- 과도한 추상화 금지
- 이해하기 쉬운 코드 작성

## Behavior
- 코드를 생성하기 전 구조를 먼저 설명
- 여러 파일이 필요한 경우 구조 먼저 제안
- 한국어로 설명 후 코드 생성
